用Github制作个人博客主页

2020-12-28

Github是一个免费的代码托管平台，适用于大多数刚刚踏入代码界门槛的新手程序员（像我一样的），对于我而言，可能我没事闲的写的绝大多数程序代码都是不具备给别人用或者出售盈利的。就算是给我自己用，也需要借助服务器发布才能访问，Github就可以作为轻量级的小服务器测试使用。

注册Github

  1. 访问Github官网
  2. 点击右上角 Sign in
  3. 注册Github账号
  4. 注册用QQ邮箱或者163邮箱就行，密码需要大写字母+小写字母，比如Qq.123456
    PS：注册完需要去邮箱点一下链接进行验证

使用Github搭建个人博客主页

Create a new repository 创建一个新项目库

  1. 创建个人Github Pages页面的项目名是固定的，你的注册名.github.io
  2. 仓库选择Public公共（免费无限制），反正你的博客也是给人家随便看的
  3. 勾选Add a RADEME file，这是创建一个项目说明文件使项目初始化
  4. 拉到下面Creat创建就行了

生成你的博客主页

  • 会搞代码的自己去网上扒个模板下来改改也行
  • 不会搞代码的就用Hexo，Gridea，Typora这些MD编辑器
  • 个人推荐博客用Gridea做基础架构
  • 偶尔写文用Typora，然后复制到Gridea就行
  • Hexo部署还是需要一定的基本功
  • Gridea的界面不算负责，我大概几分钟就玩了大部分功能
  • 这种是静态博客搭建，如果需要帮助，可以去我的动态博客留言
